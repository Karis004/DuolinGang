# DuolinGang
A Cantonese Learning Website

## Features

- Flashcard memorization and review
- Definitions and example sentences from words.hk
- Cantonese pronunciation support
- User authentication system
- Personal vocabulary management

## Tech Stack

- React, Next.js
- Tailwind CSS
- Web Speech API
